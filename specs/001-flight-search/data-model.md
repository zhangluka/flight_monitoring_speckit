# Data Model: 国内航班与价格查询

**Feature**: 001-flight-search  
**Date**: 2025-03-04

本文档描述规格中的核心实体、字段与校验规则，不绑定具体实现技术。

---

## 1. 搜索条件（SearchParams）

用户发起查询时提交的条件。

| 字段                  | 说明                | 校验规则             |
| --------------------- | ------------------- | -------------------- |
| 出发地（origin）      | 城市或机场代码/名称 | 必填；与目的地不同   |
| 目的地（destination） | 城市或机场代码/名称 | 必填；与出发地不同   |
| 出行日期（date）      | 出发日期            | 必填；不能为过去日期 |

- **粒度**：出发地、目的地支持「城市」或「具体机场」；选城市时，后端/API 返回该城市所有相关机场的航班。
- **身份与唯一性**：无持久化；每次请求独立，无唯一 ID。

---

## 2. 航班（Flight）

单条航班信息，来自第三方 API 经 BFF 归一化后的结构。

| 字段                      | 说明                      | 备注                   |
| ------------------------- | ------------------------- | ---------------------- |
| 航班号（flightNumber）    | 如 CA1234                 | 必现                   |
| 起飞时间（departureTime） | 出发时刻                  | 统一时区（如北京时间） |
| 降落时间（arrivalTime）   | 到达时刻                  | 统一时区               |
| 出发地（origin）          | 机场或城市代码/名称       | 与搜索条件对应         |
| 目的地（destination）     | 机场或城市代码/名称       | 与搜索条件对应         |
| 舱位与价格（cabins）      | 至少一项：舱位名称 + 价格 | 见下节                 |

- **关系**：航班与「舱位」为一对多（一条航班可有多舱位）；列表展示时至少展示一种舱位与价格（规格约定）。

---

## 3. 舱位（Cabin）

与航班关联的舱位及其价格。

| 字段             | 说明             | 备注                     |
| ---------------- | ---------------- | ------------------------ |
| 舱位名称（name） | 如经济舱、公务舱 | 展示用                   |
| 价格（price）    | 金额 + 货币      | 规格要求统一为人民币展示 |

- **与航班关系**：多条舱位属于一条航班；列表至少展示一种舱位与价格。

---

## 4. 状态与无持久化

- **加载中**：已提交搜索、等待 BFF/第三方 API 返回期间的 UI 状态。
- **无结果**：API 返回空列表时，展示「无结果」提示。
- **错误**：网络或服务异常时，展示可理解的错误提示；不暴露技术堆栈。
- 本功能**不持久化**用户数据；不涉及用户表、历史记录表等。

---

## 5. 与规格的对应

- **FR-001**：搜索条件 = 出发地、目的地、日期；出发地/目的地支持城市或机场。
- **FR-002**：结果列表 = 航班列表，每条含航班号、起降时间、舱位与价格。
- **FR-005**：搜索条件校验 = 必填、出发地≠目的地、日期非过去。
